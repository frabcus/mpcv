Archives are zip files containing HTML index pages, thumbnail images and the CV documents themselves.

After downloading an archive, first unzip it. Then open the index file which is here:

You need to be connected to the Internet to view the CVs using the in-page viewer. You can however download them, or browse them as files locally.
